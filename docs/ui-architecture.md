### 选择 Tauri 的理由

1. **Rust 同构**: 核心分析引擎用 Rust 写，UI 后端也 Rust — 零 FFI 开销直接调用
2. **体积优势**: 整个应用 < 15MB（含前端），对比 Electron 的 150MB+
3. **内存优势**: 系统级的内存占用，适合作为常驻维护工具
4. **Web 前端灵活性**: 用 React/Svelte 构建 UI，生态丰富，组件库多
5. **Tauri v2**: 2024 年稳定版，支持移动端 (iOS/Android)，为未来扩展留空间

### 前端技术选型

| 选项 | 推荐度 | 理由 |
|------|--------|------|
| **React + shadcn/ui** | ⭐⭐⭐⭐⭐ | 生态最丰富，组件质量高，TypeScript 支持好 |
| Svelte 5 | ⭐⭐⭐⭐ | 体积更小，响应式原生，学习曲线低 |
| Vue 3 + Naive UI | ⭐⭐⭐ | 中文社区强，但对 Tauri 的适配不如 React |
| Vanilla HTML/JS | ⭐ | 不推荐，缺少组件体系 |

**推荐: React 18 + TypeScript + shadcn/ui + Tailwind CSS**
