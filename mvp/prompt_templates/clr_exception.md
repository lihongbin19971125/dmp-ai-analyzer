## 分析重点: CLR/.NET 异常 (E0434352 / E0434F4D)
常见根因:
1. **未处理异常** — 托管代码中未捕获的异常
2. **GC 压力** — 托管堆碎片、LOH 碎片、GC 模式问题
3. **P/Invoke 问题** — 非托管代码调用导致的栈不平衡或内存损坏
4. **COM Interop** — 托管/非托管边界问题
5. **Finalizer 阻塞** — 终结器线程长时间运行

## 分析要求
### 1. 崩溃摘要 — 包含 HRESULT 和异常消息
### 2. 根因分析:
- 托管调用栈 (如果有) 中的异常类型
- GC 模式 (Server/Workstation) 和堆状态
- 是否有 P/Invoke 调用在崩溃附近
### 3. 修复建议
### 4. 置信度
